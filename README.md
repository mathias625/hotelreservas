# Hotel Reservas

## IDE Utilizada

Visual Studio Code

## Banco de Dados

MySQL

## Servidor

Node.js

## Tecnologias

- HTML
- CSS
- JavaScript
- Express
- Prisma
- MySQL

## Como Executar

1. Criar banco hotel_db

2. Instalar dependências

npm install

3. Gerar Prisma

npx prisma generate

4. Executar migrações

npx prisma migrate dev

5. Iniciar API

npm run dev

6. Abrir index.html

## Funcionalidades

- Cadastro de quartos
- Exclusão de quartos
- Cadastro de reservas
- Exclusão de reservas
- Visualização de reservas por quarto
